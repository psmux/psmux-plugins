# psmux-theme-onedark

> Atom's iconic One Dark theme for psmux.

Clean, modern dark theme with vibrant accent colors. Features powerline segments, Nerd Font icons, and multiple separator styles.

## Installation

```
set -g @plugin 'marlocarlo/psmux-plugins/psmux-theme-onedark'
```

## Options

```bash
set -g @onedark-show-powerline 'on'
set -g @onedark-separator 'arrow'       # arrow|rounded|slant
set -g @onedark-show-icons 'on'
set -g @onedark-show-user 'on'
```
